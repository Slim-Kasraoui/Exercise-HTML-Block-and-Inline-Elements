Instructions

In this web page, there are two <section> elements: one with a block-elements id and another with a inline-elements id.

You must fill these two sections with the most common block and inline HTML elements respectively. You will find the list of these block/inline elements as HTML comments in the source code.

For each block element that you place inside the section, you must add the class name block to it, along with an id that is made up of the element name along with the -element prefix. For example, for the article element the id must be article-element. You must also do the same for each inline element, but the class name should be inline.

The content of each element must be a text that describes the Tag used and its type enclosed in parentheses, e.g. Article Tag (Block Element)

For each element, you must add a unique CSS color (taken from the 140 available CSS color names) as a background. You must use each element's unique ID to target the element and apply the background color property.

You can find two examples for a block and an inline element already created to help you get started with this exercise.

Before starting this exercise, please move the instructions section at the bottom of the page.

Good luck!
